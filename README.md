# Android Multi-Thread Downloader

- 由于原作者不做维护了，我在基于上修改部分代码
- 喜欢用HttpURLConnection可以考虑看看这个

# 注意
- 本库不会校验文件是否正确，只会比较大小是否一致
- 需要依赖第三方开发者，在下载前和下载后，对文件做MD5或者crc32校验
